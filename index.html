// Manage Tracks
Tracks drive wind headwind/tailwind math. Set the track heading in degrees (direction cars travel). Use your phone's compass app to verify.
// Wind Direction Math
• Wind direction is stored as compass degrees
• App calculates headwind/tailwind component using track heading
Example: Redding runs North→South (180°)
A Southerly wind (180°) = direct headwind
A Northerly wind (0°) = direct tailwind
An Easterly wind (90°) = pure crosswind
Effective headwind component:
cos(wind_deg − track_deg) × wind_speed
+ = headwind (slower ET)
− = tailwind (faster ET)
